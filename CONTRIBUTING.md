# Contributing to dfxpy

First off, thank you for considering contributing to **dfxpy**! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Use the **GitHub Issues** tab.
- Describe the bug in detail and provide a minimal reproducible example (code snippet).
- State your OS and Python/Pandas versions.

### Suggesting Enhancements
- Open an issue and explain why the feature would be useful.
- If you have a design in mind, feel free to share it.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows **PEP8**.
3. Add tests for any new functionality.
4. Ensure all existing tests pass: `pytest tests/test_core.py`.
5. Update documentation if needed.

## Development Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/sayantancodex/dfxpy.git
   cd dfxpy
   ```
2. Install dependencies:
   ```bash
   pip install -e .
   pip install pytest
   ```
3. Run tests:
   ```bash
   pytest
   ```

## Code of Conduct
Please be respectful and professional in all interactions. We aim to foster a welcoming and inclusive environment.
