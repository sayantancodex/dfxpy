# Security Policy

## Supported Versions

Currently, the following versions of **dfxpy** are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

We take the security of **dfxpy** seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, follow these steps:

1.  **Email us**: Send a detailed report to `sayantanpatra68@gmail.com`.
2.  **Wait for a response**: We aim to acknowledge receipt of your report within 48 hours.
3.  **Coordination**: We will work with you to understand the scope of the issue and provide a fix.
4.  **Public Disclosure**: We will coordinate a public disclosure date once a fix is released.

Please provide as much information as possible, including:
- Steps to reproduce the vulnerability.
- Potential impact.
- Any suggested fixes or mitigations.

Thank you for helping keep **dfxpy** secure!
